# The Developer Community (TDC) 🚀

Welcome to the TDC Platform—the core hub for the Developer Community project management, member onboarding, and administrative operations.

## Overview

This application is a robust, full-stack platform built with **React Router 7 (SSR)** and **Supabase**. It provides a seamless experience for developers to apply for projects, manage their profiles, and for administrators to oversee the community's growth.

### Key Features

- 👤 **Member Profiles** - Personal dashboards for developers to showcase their skills, XP, and project history.
- 📁 **Project Management** - A centralized location for browsing and applying for community projects.
- 🛡️ **Admin Dashboard** - Powerful tools for administrators to review applications, manage members, and monitor audit logs.
- 🔐 **Secure Authentication** - Built-in secure session management via Supabase Auth and HttpOnly cookies.
- 🎨 **Modern UI** - A high-performance, dark-mode first design system using CSS Modules and Radix UI.

## Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/VK-Amogh/TheDeveloperCommunity-TDC-.git
   cd TheDeveloperCommunity-TDC-
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up your environment variables in a `.env` file:
   ```env
   SUPABASE_PROJECT_URL=your_project_url
   SUPABASE_ANON_KEY=your_anon_key
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- **Framework**: React Router v7 (SSR)
- **Database / Auth**: Supabase
- **UI Components**: Radix UI
- **Styling**: CSS Modules + Custom Properties
- **Icons**: Lucide React
- **Type Safety**: TypeScript
